Fruki

Fruki reads a SAN description file and then outputs an image of the corresponding Markov Chain. The graphical generation uses the graphviz (www.graphviz.org) software package.

This is beta code (C# for MSVC2010), documentation is and more testing need.

For SAN language syntax, see it's grammar SAN.g and the samples.

Sample results

Sample One

SAN file:

autonomus X {
0 - s - 1;
1 - s - 2;
1 - u*0.7 - 2;
2 - t - 1;
1 - u*0.3 - 0;
}

autonomus Y {
3 - v - 4;
4 - s - 3;
4 - w - 3;
}

autonomus Z {
5 - v - 6;
6 - r - 5;
}

event r loc := 4;
event s syn := 5;
event t loc := Y = 4;
event u loc := 10;
event v syn := (X !> 0)*6 + (X = 0)*8;
event w loc := 2;

Sample Two

SAN file:

autonomus A {
0 - r - 1;
1 - t - 0;
1 - s - 0;
1 - r - 2;
2 - s - 0;
2 - t - 1;
}

autonomus B {
0 - w - 1;
0 - x - 2;
0 - y - 2;
1 - s - 0;
2 - v - 4;
3 - t*0.4 - 2;
3 - t*0.6 - 0;
3 - s - 0;
4 - s - 0;
4 - w - 3;
}

autonomus C {
0 - w - 1;
1 - y - 3;
1 - w - 2;
1 - y - 3;
2 - w - 3;
3 - w - 0;
3 - y - 1;
3 - r - 1;
}

event r loc := B + 10;
event s syn := 5;
event t syn := ((C = 1) or (C = 3))*2;
event v loc := (A < C)*3;
event w syn := 8;
event x loc := 9;
event y loc := 7;
